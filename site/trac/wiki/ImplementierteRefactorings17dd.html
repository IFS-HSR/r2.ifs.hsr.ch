Bevor wir mit einfachen Refactorings beginnen, implementieren wir einige praktische codegenerierende Funktionen. Diese sind im eigentlichen Sinne keine Refactorings, da effektiv Funktionalität zu einer bestehenden Applikation hinzugefügt werden kann. Bei den folgenden nicht-Refactorings liessen wir uns von den bereits in Eclipse für Java verfügbaren Funktionen inspirieren.

Geplante nicht-Refactorings:

  • Generate Getters and Setter
  • Generate Constructor Using Field
  • Add Constructor from Superclass
  • Override / Implement Methods

Geplante Refactorings:

  • Convert Local Variable to Field
  • Repace Temporary Variable with Query
  • Introduce Factory?
  • Move Field
  • Move Method
  • Move Class?
  • Extract Method
  • Extract Class
  • Method Integrate Method?
  • Class Integrate Class?
  • Rename Field
  • Rename Method
  • Rename Class
