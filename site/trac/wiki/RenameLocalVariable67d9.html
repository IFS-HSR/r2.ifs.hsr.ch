Name

Rename Local Variable

Beschreibung

Ändert den Namen einer lokalen Variablen, oder eines Methodenparameters.

Vorgehen

  1. Der Name des Feldes wird geändert.
  2. Jede Referenz auf die Variable innerhalb der Methode muss geändert werden.

Abhängigkeiten

  • Es darf kein Feld mit dem neuen Namen existieren.
